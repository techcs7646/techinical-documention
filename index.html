Introduction

JavaScript is a versatile programming language used for dynamic web development.

Setup

Follow these steps to set up your environment:

npm install cool-package

JavaScript Syntax

The syntax of JavaScript is simple and easy to learn:


let x = 10; // Declare a variable
console.log(x); // Output the value of x

JavaScript Comments

Use comments to make your code readable:


// This is a single-line comment
/* This is a multi-line comment */

Variables & Data Types

Learn about JavaScript variables and their types:


let name = "John"; // String
let age = 25; // Number
let isAdmin = false; // Boolean

Operators

JavaScript supports various operators:


let sum = 10 + 5; // Addition
let isEqual = 10 == "10"; // Comparison

Functions

Functions are blocks of code that can be reused:


function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Alice")); // Output: Hello, Alice

Scope

Scope determines the accessibility of variables:


let globalVar = "I am global";

function testScope() {
  let localVar = "I am local";
  console.log(globalVar); // Accessible
  console.log(localVar); // Accessible
}

console.log(localVar); // Error: not defined

Asynchronous JS

Understand promises, async/await, and callbacks:


fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

Modules

Learn how to structure your JavaScript applications:


// In module.js
export function greet(name) {
  return `Hello, ${name}`;
}

// In main.js
import { greet } from './module.js';
console.log(greet("Alice"));
